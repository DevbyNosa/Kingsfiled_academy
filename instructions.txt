Core Features (Must Have)
Feature	What they actually do
Enter Scores	Input test and exam scores for their subject(s)
View Students	See list of students in their class(es)
Post Assignments	Give homework, set due dates, attach files
View Submissions	See which students submitted, download their work
Grade Assignments	Mark submitted homework, give scores

Send Announcements	Post notifications to students in their class
Track Attendance	Mark which students were present

Calculate Positions	Generate term totals, averages, rankings
Publish Results	Make results visible to students
View All Subjects	See complete class performance
Add Remarks	Write comments on student reports

Action	Why
See other classes' students	Not their class
Change student passwords	Only student or admin
Delete results	Admin only
Promote students	Admin only
See salary info	HR only
Edit other teachers' scores	Only their own subject
